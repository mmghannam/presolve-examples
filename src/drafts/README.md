# Unreviewed drafts

The chapters listed below have **not yet been reviewed**. Each one was scaffolded automatically — the math, instance, and walkthrough are unchecked.

In the main book table of contents (sidebar), unreviewed chapters appear in their normal section position as greyed-out placeholders that cannot be clicked. The links in this "Unreviewed drafts" part of the sidebar give you a clickable entry point so you can read each draft and give feedback.

## Workflow

We review one chapter at a time:

1. Pick a draft from the list (or the section that's next in the book).
2. Read the existing draft prose and discuss what needs fixing.
3. Build an `instance.lp` (and `expected.lp`) for the technique.
4. Replace the draft with the final chapter.
5. Update `SUMMARY.md`: promote the in-section placeholder to a real link, and remove the entry from the "Unreviewed drafts" list.

Each draft chapter starts with a `> **DRAFT — unreviewed.**` banner under its title — that banner disappears when the chapter is promoted.
