> Paper §3.6

Consider an equation $A_{i\cdot}x = b_i$ that satisfies
$$\inf\{A_{i\cdot}x\} + \sup\{A_{i\cdot}x\} = 2 b_i$$
(after complementing variables to make all coefficients non-negative). This says the right-hand side sits exactly midway between the row's minimum and maximum activity. Now suppose $x_k$ is a binary variable in the support with the *largest* coefficient, satisfying $|a_{ik}| = \sup\{A_{i\cdot}x\} - b_i$. Then the equation forces

- $x_k = 0 \;\Longrightarrow\; x_j = u_j$ for every other $j$ in the support, and
- $x_k = 1 \;\Longrightarrow\; x_j = \ell_j$ for every other $j$ in the support.

Combining the two implications produces the explicit substitution
$$x_j := u_j - (u_j - \ell_j)\, x_k \quad \text{for all } j \in \operatorname{supp}(A_{i\cdot}) \setminus \{k\}.$$
The equation is consumed and every other variable in it disappears, replaced by an affine function of the single binary $x_k$.

Paper Example 3 — $10 x_1 + x_2 + \ldots + x_{11} = 10$ with $x_1 \in \{0,1\}$ and $x_j \in [0, 1]$ for $j = 2, \ldots, 11$ — is a clean instance: $x_1 = 0$ forces all other $x_j = 1$, and $x_1 = 1$ forces all other $x_j = 0$, giving the substitution $x_j := 1 - x_1$.
