# 4.3 Strengthen semi-continuous and semi-integer bounds

> Paper §4.3

A **semi-continuous** variable takes either the value 0 or any value in a specified interval $[\ell_j', u_j]$ with $\ell_j' \geq 0$. A **semi-integer** variable adds an integrality requirement on top.

Two reductions follow from external evidence about the variable's range:

- If presolve proves $x_j \geq \ell_j' > 0$ from the surrounding constraints, the value 0 is impossible, and $x_j$ becomes a regular continuous (or integer) variable on $[\max\{\ell_j, \ell_j'\}, u_j]$.
- If presolve proves $x_j < \ell_j'$, the only option in the disjunction is $x_j = 0$, so the variable is fixed.

Additionally, a semi-continuous variable with $\ell_j' = 0$ is just a regular variable: the "semi" property says nothing.
