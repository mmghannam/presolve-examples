> Paper §4.4

Suppose all constraints are $\leq$ inequalities or equations and $x_j$ does not appear in any equation, i.e. $a_{ij} = 0$ for every $i$ with $\circ_i = \,=$. **Dual fixing** says:

- If $c_j \geq 0$ and $a_{ij} \geq 0$ for every constraint, then making $x_j$ smaller never hurts: in any optimal solution $x_j$ can be set to $\ell_j$. (If $\ell_j = -\infty$ and $c_j > 0$ the model is unbounded.)
- The mirror case with $c_j \leq 0$ and $a_{ij} \leq 0$ everywhere fixes $x_j$ to $u_j$.

The reasoning is purely about *signs*: pushing $x_j$ in the favorable direction of the objective can only help feasibility on every row in which it appears, so an optimal solution at the bound always exists.

A useful refinement applies when there is **exactly one** "wrong-sign" row $r$ with $a_{rj} < 0$ (still in the $c_j \geq 0$ case) and a binary $x_k$ in the same row whose value already determines whether row $r$ is tight. Then $x_j$ can be substituted as an affine function of $x_k$ rather than fixed.

In the general case where dual fixing does not directly fix $x_j$, the same sign argument produces a tighter *bound*: any value $\tilde u_j < u_j$ for which the wrong-sign rows would still be satisfiable is a valid replacement upper bound.
