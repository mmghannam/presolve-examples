# 4.1 Removal of fixed variables

> **DRAFT — unreviewed.** This chapter is a scaffolded starting draft. The math, instance, and prose have not been reviewed. See the §3.5 and §4.5 chapters elsewhere in the book for the target format.

> Paper §4.1

A variable $x_j$ with $\ell_j = u_j$ is **fixed**: every feasible point assigns it the same value. Such a variable can be removed from the model by

- subtracting $A_{\cdot j} \ell_j$ from the right-hand side $b$, and
- folding $c_j \ell_j$ into the objective constant.

After the substitution $x_j$ no longer appears in any constraint or in the objective, so its column is deleted. In post-processing, recovering $x_j$ is trivial — it is fixed.

Fixed variables arise both directly in the input (e.g., a binary variable forced by data to 0 or 1) and as a *consequence* of other reductions, especially bound strengthening (§3.2) collapsing $[\ell_j, u_j]$ to a singleton.
