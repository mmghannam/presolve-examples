> Paper §6.4

In a model with all $\leq$ constraints, $x_j$ **dominates** $x_k$ (written $x_j \succ x_k$) if

1. $c_j \leq c_k$ (using $x_j$ is no more expensive than using $x_k$),
2. $a_{ij} \leq a_{ik}$ in every row (using $x_j$ never makes any $\leq$ constraint harder than using $x_k$ would),
3. $j \in N \setminus I$ or $k \in I$ (the integrality types are compatible — a continuous variable cannot be dominated by an integer one without losing options).

Domination supports several fixings (with infinite-bound preconditions):

- If $u_j = \infty$ and $x_j \succ x_k$, then $x_k$ can be set to $\ell_k$. (Anywhere $x_k$ was used, swap to $x_j$ instead — feasible and no worse for the objective.)
- If $u_j = \infty$ and $x_j \succ -x_k$ (the complement), then $x_k$ can be set to $u_k$.
- Symmetric cases with $\ell_k = -\infty$ go the other direction.

The infinite-bound requirement looks restrictive but the implied-free check (§4.5) often supplies effective infinity: if no other constraint needs $u_j$ to bound $x_j$, the explicit bound can be removed without losing feasibility.
