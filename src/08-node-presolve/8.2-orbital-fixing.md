# 8.2 Orbital fixing

> Paper §8.2

If a model has a symmetry group $G$ — variable permutations that leave the model invariant (§7.1) — then for any feasible solution there are equivalent feasible solutions obtained by applying group elements. Branching that fixes one variable in a $G$-orbit creates a node at which the symmetry is partially broken, but the residual stabilizer subgroup still acts on the remaining variables.

**Orbital fixing** uses the residual symmetry to tighten the local node. For an orbit $O$ under the residual group, any feasible point can be permuted so that the largest variable in $O$ lies at any chosen index. In particular, the upper bound of every variable in $O$ can be tightened to
$$x_j \leq \min\{\bar u_k : k \in O\} \qquad \text{for all } j \in O.$$
A symmetric argument tightens lower bounds, though the bookkeeping is more involved.

Computing the residual symmetry group at every node is expensive. Cheap alternatives:

- **Filter the global generators.** Discard generators of $G$ that no longer respect the local lower bounds; the remaining generators generate a *subgroup* of the residual stabilizer, which is enough to derive valid orbital fixings.
- **Recompute from scratch at the node.** Slower but exploits symmetry that the global group missed (e.g., symmetry introduced by branching).

Orbital fixing applies only to a subset of models — those with measurable symmetry — but on those models the per-node savings are large.
