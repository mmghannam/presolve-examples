> Paper §8.4

Probing (§7.2) at the root only probes binary variables that are *fractional* in the root LP — variables that came out integer at the root are not probed, since there is no immediate need to refine them. As the search progresses and reduced-cost fixing or branching changes which variables are fractional, additional binaries become candidates for probing. **Global probing at local nodes** runs probing on these fresh candidates, using globally valid bounds (not the local node's bounds) to deduce globally valid implications.

The reductions are deliberately conservative. Unlike root probing, the local-node version

- only collects implied bound information (no variable substitution, since substitutions would invalidate stored warm-start LP bases at sibling nodes),
- only probes a binary the first time it becomes fractional during the search (each binary is probed at most once globally),
- uses the current global bounds rather than local ones — the conclusions are valid throughout the entire tree, not just below the current node.

The output of global probing is exactly the same as root probing: globally valid bound changes, fixings, implications added to the implication graph, and statistics for branching variable selection. The difference is timing — these reductions are deferred to the moment they become available, rather than lost because the root LP did not flag the relevant variable.
