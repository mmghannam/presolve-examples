> Paper §7.4

A **biconnected component** is a generalization of disconnected components (§7.3): instead of two pieces that share *no* variable, the model decomposes into two pieces that share exactly *one* variable — an **articulation variable** $x_k$. Fixing $x_k$ disconnects the rest.

Concretely, the model
$$\min c_U^T x_U + c_V^T x_V + c_k x_k \quad \text{s.t.} \quad A_{RU} x_U + A_{Rj} x_k \leq b_R,\ A_{SV} x_V + A_{Sj} x_k \leq b_S$$
splits, after fixing $x_k = \bar{x}_k$, into the two independent sub-MIPs
$$\min\{c_U^T x_U + c_k \bar{x}_k : A_{RU} x_U \leq b_R - A_{Rj} \bar{x}_k\}, \quad \min\{c_V^T x_V : A_{SV} x_V \leq b_S - A_{Sj} \bar{x}_k\}.$$

If $x_k$ is binary, this is exactly probing (§7.2): solve both sub-MIPs at $x_k = 0$ and $x_k = 1$, and harvest

- a fixing if one branch is infeasible, or
- a substitution if both branches yield the same value of $x_j \in V$, or
- a globally valid implication.

The sub-MIPs are smaller than the original, so the cost of probing the articulation variable is dominated by the savings of decomposing.
