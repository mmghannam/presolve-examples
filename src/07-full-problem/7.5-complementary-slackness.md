# 7.5 Exploiting complementary slackness

> Paper §7.5

For the LP $\min\{c^T x : Ax \geq b,\ x \geq 0\}$ and its dual $\max\{b^T y : A^T y \leq c,\ y \geq 0\}$, optimality requires **complementary slackness**:
$$x_j^\star > 0 \;\Longrightarrow\; c_j - (A_{\cdot j})^T y^\star = 0, \qquad y_i^\star > 0 \;\Longrightarrow\; A_{i\cdot} x^\star - b_i = 0.$$

For a MIP $\min\{c^T x : Ax \geq b, \ell \leq x \leq u, x_j \in \mathbb{Z} \text{ for } j \in I\}$ with $\ell = 0$, although there is no general MIP duality, complementary slackness on the *continuous* part of the model can still produce useful reductions:

- Apply bound strengthening (§3.2) to the dual constraints $(A_{\cdot S})^T y \leq c_S$ over $S = N \setminus I$ to derive bounds $\bar\ell \leq y \leq \bar u$.
- For any row $i$ with $\bar\ell_i > 0$, the dual variable is strictly positive in every optimal solution, so by complementary slackness the primal row is tight: $A_{i\cdot} x = b_i$. The inequality can be replaced by an equation.
- For any column $k$ with $c_k > \sup\{(A_{\cdot k})^T y : \bar\ell \leq y \leq \bar u\}$, the dual feasibility constraint is strictly slack at every dual-feasible $y$, so by complementary slackness $x_k^\star = 0$. The variable is fixed at its lower bound.

These reductions transfer information from the LP relaxation's dual into the primal MIP — variable fixings and tight equations that hold for *every* optimal solution.
