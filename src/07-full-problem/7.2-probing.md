**Probing** tentatively sets a binary variable $x_k$ to 0 and to 1, runs domain propagation under each assumption, and harvests the consequences. Let $\ell_j^0, u_j^0$ be the bounds on $x_j$ deduced from $x_k = 0$ (via §3.2 propagation), and $\ell_j^1, u_j^1$ those from $x_k = 1$. Four kinds of conclusion follow:

1. **Infeasibility-driven fixing.** If propagating $x_k = 0$ produces infeasibility, then $x_k = 1$ in every feasible solution, so $x_k$ is fixed. Symmetric for the other case.
2. **Implied substitution.** If $\ell_j^0 = u_j^0$ and $\ell_j^1 = u_j^1$ for some $j$, then $x_j = \ell_j^0 + (\ell_j^1 - \ell_j^0) x_k$ — $x_j$ is an affine function of $x_k$ and can be substituted out.
3. **Tighter global bounds.** If both probes return finite bounds for $x_j$, then $\ell_j := \min\{\ell_j^0, \ell_j^1\}$ and $u_j := \max\{u_j^0, u_j^1\}$ are valid global bounds.
4. **Implications.** Anything weaker than the above is recorded as the implication $(x_k = 0 \Rightarrow \ell_j^0 \leq x_j \leq u_j^0)$, used downstream by branching, separation of implied bound cuts, and propagation at nodes.

A useful refinement is **lifting**: if probing finds an implication $x_k = 1 \Rightarrow s \geq d > 0$ on the slack variable of an inequality $A_{i\cdot} x \leq b_i$, then the inequality can be replaced by $A_{i\cdot} x + d\, x_k \leq b_i$, which dominates the original and is tighter in the LP.

Probing is sequence-dependent (the propagation under one probe uses the bounds tightened by previous probes), so the order in which candidate variables are probed matters. Heuristics order candidates by the number of binary fixings they directly imply.
