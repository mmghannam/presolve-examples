The mixed integer program

Given a matrix A∈Rm×n, vectors c∈Rn and b∈Rm, lower bounds ℓ∈(R∪{−∞})n and upper bounds u∈(R∪{∞})n, an integer index set I⊆N={1,…,n}, and relations ∘i​∈{=,≤,≥} for each row i∈M={1,…,m}, the mixed integer program is

min s.t. ​cTxAx∘bℓ≤x≤uxj​∈Zfor all j∈I.​

We write Ai⋅​ for row i, A⋅j​ for column j, and supp(Ai⋅​)={j∈N:aij​=0} for the support of a row.

Row activities

The minimal and maximal activity of row i are

inf{Ai⋅​x}=j:aij​>0∑​aij​ℓj​+j:aij​<0∑​aij​uj​,sup{Ai⋅​x}=j:aij​>0∑​aij​uj​+j:aij​<0∑​aij​ℓj​,

with the conventions s⋅∞=∞ for s>0 etc. These activities are the workhorse quantities of single-row presolve.

Tolerances

The feasibility tolerance ε controls when an inequality is considered redundant or infeasible; in Gurobi, ε=10−6 by default. The constant ψ=1030 is the convention for "infinity" in the model data.

Conversion between ≤ and ≥ is by negation, so without loss of generality the descriptions in this book consider only one direction of each inequality.
